2. java war 打包、解压命令 : 

```

	// 将当前目录打包成war包

	jar  -cvf  temp.war  *

	命令格式：

	java cvf 打包文件名称 打包当前目录中的所有文件

 

	解压自然就是:

	jar xvf temp.war


```
